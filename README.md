# 🧠 CNN Image Classification

## 📌 Description
This project uses a Convolutional Neural Network (CNN) to classify images.

## ⚙️ Tech Stack
- Python
- TensorFlow
- OpenCV
- NumPy

## 🚀 Features
- Image classification using deep learning
- Good accuracy model
- Easy to run

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python main.py
