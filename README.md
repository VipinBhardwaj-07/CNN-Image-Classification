# 🐱🐶 CNN Image Classification (Cat vs Dog)

A deep learning project that classifies images into **Cat** and **Dog** categories using **Convolutional Neural Networks (CNN)**.

---

## 📌 Project Overview

This project is built using CNN architecture to perform binary image classification.  
The model is trained on a Cat vs Dog dataset and predicts whether the input image belongs to a cat or a dog.

This project demonstrates:

- Image preprocessing
- Model training using CNN
- Prediction on custom images
- Performance visualization using graphs

---

## 🚀 Features

✅ Binary image classification  
✅ Image preprocessing and normalization  
✅ CNN model training  
✅ Prediction system for new images  
✅ Accuracy and loss visualization

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/VipinBhardwaj-07/CNN-Image-Classification.git
```

Go to project folder:

```bash
cd CNN-Image-Classification
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶ Usage

Run training:

```bash
python main.py
```

Run prediction:

```bash
python predict.py
```
